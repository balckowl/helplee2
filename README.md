# helplee

helpleeは、マンスリーハッカソンvol.11(2023年11月)に作成したサイトです。メンバー全員初めてのハッカソンであったにもかかわらず、パーソルキャリア賞をいただきました🥉
その後、サポーターズ主催のアドバンスハッカソン(2023年12月)にてアップデートを行いました。

- **フロントエンド**: [@y_ta](https://github.com/balckowl)、[@GO0128](https://github.com/Go-Morishita)
- **バックエンド**: [@y_ta](https://github.com/balckowl)
- **デザイン**: [@GO0128](https://github.com/Go-Morishita)、[@Nakae](https://github.com/Kosyake8581)
- **プロジェクトトリーダー**: [@y_ta](https://github.com/balckowl)

アドバンスハッカソンでは、賞をいただくことはできませんでしたが、他の参加者の方からとてもいい刺激を受けることができました!!

## 作品概要 📦
helpleeは、GUIで直感的にCSSを生成できるサイトです。ボックスシャドウジェネレータやグラデーションジェネレータなどのCSS生成サイトはあるものの、それは別々のサイトとしてあり、一度生成したお気に入りのCSSを保存することができません。
本サイトはこのような問題を解決しました。
